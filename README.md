## Описание программы (кратко)
**Movie Library** — это настольное GUI-приложение для организации личного каталога фильмов. Программа позволяет структурировано хранить данные о просмотренном кино, проводить быстрый поиск по базе и автоматически сохранять информацию между сессиями.

### Примеры использования
*   **Добавление данных:** сохранение названия, жанра, года выпуска и рейтинга фильма.
*   **Валидация:** автоматическая проверка корректности ввода (год должен быть числом, рейтинг — от 0 до 10).
*   **Фильтрация:** мгновенный поиск фильмов по жанру или году выпуска в режиме реального времени.
*   **Сохранение:** автоматический экспорт и импорт базы данных в формате **JSON**.
*   **Интерфейс:** удобная таблица для наглядного отображения всей библиотеки
